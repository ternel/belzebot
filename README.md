Installation
--------------------

1. Installation des vendors :

    ``` sh
    $ npm install
    ```

3. Configuration :

    ``` sh
    $ cp config.json-dist config.json
    ```

3. Lancement de l'appli :

    ``` sh
    $ node bot.js
    ```
